# Custom Animated Hero Block for Gutenberg

This is a high-performance, accessible Gutenberg block developed with **React**, **GSAP**, and **Tailwind CSS**.

## Key Features
- **Modern Architecture**: Built using the WordPress Block API (v2).
- [cite_start]**Smooth Animations**: Integrated with **GSAP** for engaging user experiences. [cite: 16]
- [cite_start]**Responsive & Accessible**: Follows **WCAG 2.1** standards for full accessibility. [cite: 18]
